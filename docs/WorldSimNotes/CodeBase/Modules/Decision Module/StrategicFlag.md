# Strategic Flag
Multi-tick concerns in a colony that should create action by the faction. These flags represent issues in the colony that could worsen if left unattended.


Types of flags

| Flag Name            | ID  | Description                                     |
| -------------------- | --- | ----------------------------------------------- |
| Defense Needed       | 0   | net DEFENSE score below threshold               |
| Worker Shortage      | 1   | not enough workers to staff buildings           |
| Resource Low         | 2   | one or more key resources trending negative     |
| Export Strained      | 3   | tax rate is cutting into operational reserves   |
| Construction Blocked | 4   | colony wants to build but can't afford anything |
